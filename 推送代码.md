# 推送代码到 GitHub

## 当前状态

✅ Git 仓库已初始化
✅ 代码已提交
✅ GitHub 远程仓库已连接

## 需要认证

GitHub 需要认证才能推送代码。

## 方法 1: 使用 Personal Access Token（推荐）

### 第一步：创建 Token

1. 访问：https://github.com/settings/tokens
2. 点击 "Generate new token" → "Generate new token (classic)"
3. 填写：
   - Note: `us-china-picker`
   - Expiration: 选择期限
   - 勾选：`repo` (全部权限)
4. 点击 "Generate token"
5. **复制 token**（格式：`ghp_xxxxxxxx...`）

### 第二步：使用 Token 推送

**选项 A：使用 Token 配置远程 URL（推荐）**

```bash
# 替换 YOUR_TOKEN 为实际 token
git remote set-url origin https://YOUR_TOKEN@github.com/anniezty/us-china-news-scrapper.git

# 推送
git push -u origin main
```

**选项 B：推送时输入**

```bash
git push -u origin main
```

提示时：
- Username: `anniezty`
- Password: **粘贴 token**（不是密码）

## 方法 2: 安装 GitHub CLI（更简单）

```bash
# 安装 GitHub CLI（如果还没安装）
brew install gh

# 登录
gh auth login

# 选择 GitHub.com → HTTPS → 登录

# 然后推送
git push -u origin main
```

## 方法 3: 配置 SSH（长期方案）

```bash
# 生成 SSH key
ssh-keygen -t ed25519 -C "your-email@example.com"

# 复制公钥
cat ~/.ssh/id_ed25519.pub

# 添加到 GitHub: Settings → SSH and GPG keys → New SSH key

# 改用 SSH URL
git remote set-url origin git@github.com:anniezty/us-china-news-scrapper.git

# 推送
git push -u origin main
```

## 推荐：使用方法 1（最简单快速）

创建好 token 后，在 Terminal 执行：

```bash
git remote set-url origin https://YOUR_TOKEN@github.com/anniezty/us-china-news-scrapper.git
git push -u origin main
```

**记住**：将 `YOUR_TOKEN` 替换为实际的 token！
